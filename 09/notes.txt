ARRAYS IN JAVA

arrays are non-primitive data structures
they are the list of elements of same type stored in a contiguous memory location 

> arrays in JAVA follow zero based indexing 
> the memory locations in array are contageous

array elements :    a , b , c 
memeory addresses : 1001 , 1003 , 1005

arrays are static, the memeory once defined cannot be changed at the runtime
